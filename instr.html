🧭 Навигатор автоинструктора

Как это работает

Выбираете маршрут из списка. Каждая точка — место на карте с командой. При приближении к точке приложение автоматически озвучивает команду.

Элементы управления

Кнопка «Маршруты»

Центрированная кнопка вверху. Открывает список категорий маршрутов. Выберите категорию, затем маршрут — приложение загрузит точки и включит навигацию.

Верхние круглые кнопки

🗺️
Спутник / Схема
Слева вверху

Переключает между спутниковыми снимками и схемой.

🎯
Привязка карты к текущей геопозиции
Справа вверху

Включает режим следования: карта автоматически центрируется по вашему местоположению и поворачивается по направлению движения. Повторное нажатие отключает.

2D / 3D
Наклон карты
Слева под кнопкой переключения слоев карты

Переключает вид карты между 2D (вид сверху) и 3D (наклон 60°).

▶
Симуляция проезда
Слева под кнопкой наклона карты (2D / 3D)

Запускает симуляцию проезда по пути текущей точки. Сначала воспроизводится голосовая команда, затем начинается анимация движения по маршруту. Повторное нажатие останавливает симуляцию.

Нижняя панель (HUD)

🔊
Блок команды
Центральная кнопка внизу

Показывает номер и текст текущей команды, расстояние до точки. Нажмите — приложение озвучит команду голосом.

◀ ▶
Навигация по точкам
Внизу по бокам от блока команды

◀ — предыдущая точка, ▶ — следующая. При переходе карта плавно центрируется на точке. Между кнопками появляется Выбрать — при нажатии делает просматриваемую точку текущей целью.

Карта

Точки маршрута отображаются цветными маркерами. Текущая активная точка подсвечена зелёным кольцом. Цвет маркера соответствует типу команды. Нажмите на любую точку на карте — откроется предпросмотр с командой и путём. Нажмите в любом месте карты — предпросмотр закроется.

Голосовые команды

При приближении к целевой точке (<25 м) и совпадении направления (+-45°) приложение автоматически проигрывает голосовую команду. Расстояние до точки показывается справа в блоке команды.

Комментарии

Точки с комментарием отмечены красным контуром. Комментарий отображается сверху над командой. Нажмите, на него чтобы скрыть.

Советы

  • Разрешите геолокацию для корректной работы навигации.
  • Приложение работает только при включенном экране.
  • При ограничениях работы интернета может не работать, используйте vpn.
  • В некоторых местах в определенное время могут глушить GPS и приложение не будет работать корректно.
  • Включите автоцентрирование (🎯), чтобы карта следовала за вами.
